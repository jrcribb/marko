# Render
```html
<p
  class="A"
>
  paragraph
</p>
<button />
```

# Update
```js
document.querySelector("button").click();
```
```html
<p
  class="B"
>
  paragraph
</p>
<button />
```
## Change
```
UPDATE: .B[class] "A" => "B"
```
