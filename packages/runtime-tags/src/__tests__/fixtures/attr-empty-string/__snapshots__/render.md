# Render
```html
<div
  title=""
/>
<button
  type="button"
>
  toggle
</button>
```

# Update
```js
document.querySelector("button").click();
```
```html
<div
  title="set"
/>
<button
  type="button"
>
  toggle
</button>
```
## Change
```
UPDATE: div[title] "" => "set"
```
